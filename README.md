# Ping2Bro  
Code with your friends. See who's active, chat inside VS Code.

## Features
- **Real-time Presence**: See exactly what file your friends are working on inside VS Code.
- **Friend Code System**: Add friends securely using unique 8-character Ping Codes.
- **Mutual Friends Only**: Your activity is only visible to people who have accepted your friend request.
- **Built-in Chat**: Direct message your friends without ever leaving your IDE.
- **Smart Idle Timer**: Automatically marks you as "idle" when you step away, and "offline" when you close VS Code.
- **Instant Notifications**: Get a pop-up in VS Code the moment a friend comes online.

## Getting Started
1. Click the **Ping2Bro icon** in your left activity bar.
2. Click **Sign in with Google**.
3. Once logged in, switch to the **Friends tab** to see your unique Ping Code.
4. Share your code with a friend, or enter their code to send a friend request.
5. Once they accept, you'll see them in the **Online tab** whenever they are coding!
6. Click on a friend to open a direct message **Chat**.

## Extension Settings
This extension contributes the following settings:

* `ping2bro.showNotifications`: Enable/disable popups when friends come online.
* `ping2bro.idleTimeoutMinutes`: How many minutes of inactivity before you are marked as idle.
